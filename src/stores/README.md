## ① 目的
- 本資料夾使用 Pinia 存放專案的狀態管理模組 (Stores)。

## ② 結構與命名規範
- **檔案/元件命名**：Store 檔案應使用 `use` 開頭並以 `Store` 結尾，例如 `useCartStore.js`。
- **輸出/匯入約定**：每個 Store 檔案應獨立導出一個 `defineStore` 實例。

## ③ 依賴與界面
- **相依套件/Plugin**：`pinia`
- **對外介面**：
    - `useCartStore`：管理購物車狀態、商品列表、計算邏輯。
